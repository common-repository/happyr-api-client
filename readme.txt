This is the Happyr API client. This plugin helps you to integrate the PHP library with Wordpress.

== Description ==
The plugin is a wordpress wrapper for the PHP API client for HappyRecruiting. It uses the composer.phar to
get the libraries that we depend on. Everything is included, you just need to press the install button on the
plugin's admin page.

== Installation ==

This is some brief installation notes. 

1. Upload the plugin source to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Install the vendors by clicking 'Install vendors' on the pluigns admin page
1. Add you API credentials on the admin page.
